## 🧩기능
### 1. 아이템 강화하기!
  - 다 벨 수 있는 검, 다 뚫을 수 있는 창, 다 막을 수 있는 방패 중 한 가지를 골라 강화함

### 2. 확률이 낮을수록 더 강하게!
  - 강화에 사용할 수 있는 확률 주문서는 50%(+1강), 20%(+2강), 5%(+5강)의 세 종류가 있음

### 3. 약간의 긴장감!
  - 어떤 결과가 나오는지 궁금하도록 결과 출력까지 잠깐의 딜레이
